AntiGemini — локальный AI-чат на базе gemma2:9b, работающий через Ollama без интернет-зависимости и внешних фильтров.

🚀 Возможности
Полностью локальная работа (offline AI)
Использует модель gemma2:9b через Ollama
Нет Google-фильтров и внешних ограничений
Возможность кастомизации system prompt
Простой web-интерфейс через Flask (localhost:5000)

⚙️ Установка
1. Установи Ollama
Arch Linux:
yay -S ollama
Другие системы:
Скачать можно здесь: https://ollama.com

3. Установи Python зависимости
pip install ollama flask

5. Скачай модель
ollama pull gemma2:9b

Перейди в папку проекта:

cd AntiGemini
python app.py
5. Открой в браузере
http://localhost:5000

🧠 Настройка AI
System prompt можно изменить в файле:
app.py

Это позволяет полностью менять поведение модели.

💻 Требования

GPU: GTX 1060 6GB

Python: 3.10+
Ollama установлен и настроен

Рекоминдувымая конфигурация:
